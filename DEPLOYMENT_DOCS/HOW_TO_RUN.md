# How to Run Locally

## Prerequisites
- Node.js 20+ (used in Dockerfile; no .nvmrc present)
- npm (comes with Node.js)

## Install Dependencies
```bash
npm install
```

## Environment Variables
No environment variables required. This is a static Next.js frontend with no backend services.

## Start Services
```bash
npm run dev
```

## Local URLs
- Frontend: http://localhost:3000
